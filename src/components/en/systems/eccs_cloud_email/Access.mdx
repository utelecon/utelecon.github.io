1. Access the [ECCS Cloud Email member-only test page (Google Spreadsheet)](https://docs.google.com/spreadsheets/d/16mWLhEldvrelJNJJ3oa4VlFpnMeSGYgfBja7pTpAL_s/edit?usp=sharing).

If you cannot see the contents and instead see the “You need access permission” message, follow the procedures in the “Troubleshooting guide” section below. Do not request for permission.
