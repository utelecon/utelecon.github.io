As of January 2026, this page contains information about ECCS2021, which has been in operation since March 2021. However, ECCS2021 will be gradually discontinued from January 2026 onwards, and ECCS2026 will begin operation from March 2026. Along with the end of ECCS2021 operations, **[data stored in ECCS terminals and WebDAV environment (Proself) is scheduled to be deleted on February 21, 2026](/notice/2025/1204-eccs-data-deletion/) (in Japanese)**. Please ensure that you **back up any important data before this date**.

For more details, please see [About the ECCS2026](/notice/2025/eccs2026-announcement/) (in Japanese).
