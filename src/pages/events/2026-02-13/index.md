## 開催趣旨

2024年10月から2029年9月までの5年間,Microsoft社より本学に Microsoft Azure のギフトクレジットが寄付されることになりました.
これにより,本学の構成員は Azure の各種サービスを利用できる [**「UTokyo Azure」**](/research_computing/utokyo_azure/)を,研究や教育の場で活用いただけます.

本イベントでは,UTokyo Azureの概要,Azureを活用した最新の研究事例の紹介や,Azureの代表的なサービスのチュートリアルを実施します.
また,当日はAzure利用相談ブースも設けます.

クラウド利用に関心のある方や,Azureを研究・教育で活用を検討されている方は,ぜひご参加ください.
