## Overview

The Course Materials function allows you to manage and publish the materials you need for your classes in UTOL. You can set the publication period and targets for the materials and check which students have viewed them and when.

The course materials can be managed and viewed by the course instructors, TAs and course designers. However, only the course instructors and TAs can check the viewing status of the course materials.

### ‚ÄúCourse Materials‚Äù and ‚ÄúSupplementary Course Materials‚Äù

In UTOL, all materials are managed as ‚Äúcourse materials‚Äù. Each individual material is called a ‚Äúsupplementary course materials‚Äù, and these are grouped together to form the ‚Äúcourse materials‚Äù. Settings such as the publication period and publication targets can be set for each ‚Äúcourse material‚Äù.

The ‚Äúcourse materials‚Äù can be divided into sections for each class (e.g. ‚Äú1st Lecture Materials (4/6)‚Äù) or for each group of content (e.g. ‚ÄúCriminal Procedure Law-related Materials‚Äù). Furthermore, using the ‚Äú[label function](/en/utol/lecturers/settings/#label)‚Äù, you can also manage the ‚Äúcourse materials‚Äù by attaching labels (e.g. ‚ÄúFor 1st year students‚Äù) that are common for assignments and tests.

### ‚ÄúFiles‚Äù and ‚ÄúLinks‚Äù

There are two types of teaching materials in the ‚ÄúCourse Materials‚Äù, ‚ÄúFiles‚Äù and ‚ÄúLinks‚Äù. ‚ÄúFiles‚Äù are files uploaded directly to UTOL, while ‚ÄúLinks‚Äù are links to external sites.

If you are unsure which type of teaching material to choose, consider the file size as a guide. Only files of **100MB or less** can be uploaded directly as ‚ÄúFiles‚Äù. Therefore, for larger files, you will need to upload them to an external site and provide the link as a ‚ÄúLink‚Äù. For example, you could upload class slides directly as ‚Äúfiles‚Äù, while uploading class recordings to Google Drive, YouTube, etc. and posting the links as ‚ÄúLinks‚Äù.

Even if the file size is small, you can also manage it using ‚ÄúLinks‚Äù. This is the case when you manage the materials on an external site (e.g. Google Slides).

## Managing course materials

This section explains the procedure for managing course materials (registering, updating, deleting).

### Procedure for registering course materials

To register new course materials, open the ‚ÄúCourse Top‚Äù screen and click the ![Register new course materials](../../_icons/add_coursetop.png){:.icon} button at the bottom left of the ‚ÄúCourse Materials‚Äù section.

![](./01-settings-entry.drawio.png)

The ‚ÄúCourse Materials Edit‚Äù page will open. Please fill in or select the following information (explanation will be given over two images).

![](./02-settings-01.png)

* ‚ÄúTitle‚Äù (Required)

  * Example: Lecture materials for the first class (4/6)
* ‚ÄúContents‚Äù
  * Enter a description of the course materials.
* ‚ÄúPublication period‚Äù (Required; Please check carefully)
  * Example: From the day before the start of classes to the day of the final exam
  * Start date and time of publication: To avoid a sudden increase in the load on UTOL immediately before the start of classes, it is a good idea to make the course materials available at least a day before the start of classes, so that students can download them in advance.

  * End date of publication: It is a good idea, when setting this date, to add some extra days so that students can view the materials up to the exam date or the final assignment deadline.
    * The end date of the semester is set as the default day. However, the **exam date and assignment deadline for some subjects may be later than the end date of the semester (especially in the A semester and A2 term)**, so please check the schedule carefully and set the date accordingly.

![](./03-settings-02.png)

* <DisplayLabel />
* ‚ÄúCourse Materials‚Äù
  * ‚ÄúType‚Äù

    * Select the type of course material (file or link).
  * File or link (depending on the type of course material)
    * Please add the course material according to the following procedure.
      * In the case of a file
        * Click the ‚ÄúReference‚Äù button to select a file, or drag and drop it into the specified area, and **making sure to click the ‚ÄúAdd File‚Äù button**.
      * In the case of a link
        * Enter the link and click the ‚ÄúAdd‚Äù button.

    * The file name or link name of the course material you have just added will be displayed at the bottom of the screen, so enter the ‚ÄúTitle‚Äù and ‚ÄúComments‚Äù for that material.
  * You can add multiple course materials by repeating the two steps mentioned above.
    * Please note that you can only add up to 30 files at one time.

  * If you want to delete a course material that you have already added, click the button in the ‚ÄúActions‚Äù section at the bottom of the screen and select ‚ÄúDelete‚Äù.

* ‚ÄúPublished target‚Äù
  * Select who you want to make the materials available to. You can choose from the following three types.
  * ‚ÄúAll Enrolled Students‚Äù
  * ‚ÄúUser Group‚Äù
    * User groups must be registered in advance.

    * Related:
      * [Using user group function in UTOL (for Course Instructors / TAs)](/en/utol/lecturers/settings/user_groups/)

    * Click the group name to see the enrolled students who belong to the group.
  * ‚ÄúSelect Enrolled Student(s)‚Äù

    * To make the content available to specific students, you can select them individually from the list of enrolled students.

    * A list of enrolled students will be displayed, so check the boxes of the students who you want to make the material available to.

When you have finished filling in, click the ‚ÄúConfirm‚Äù button to proceed.

The ‚ÄúCourse Materials Confirm of registration‚Äù page will open, so please check the details and click the ‚ÄúRegister‚Äù button.

![](./04-settings-confirm.png)

### Procedure for updating and deleting course materials

1. Open the ‚ÄúCourse Top‚Äù screen and click the ![edit](../../_icons/edit.png){:.icon} button to the right of the course material name in the ‚ÄúCourse Materials‚Äù section.

    ![](./05-edit-entry.drawio.png)

2. This will take you to the Course Materials Edit screen, where you can update or delete the contents.

    * To update: Edit the course material in the same way as you did when registering the course materials.
    * To delete: Click the ‚ÄúDelete course materials‚Äù button at the top right of the Course Materials Edit screen.

## Viewing Course Materials

This section explains how to view registered course materials.

Not only enrolled students, but also instructors, TAs and course designers can view registered course materials. This means that when you actually teach the class, you can display images, slides, resumes, etc. on UTOL without having to search for the files on your device. It is also useful for checking how registered course materials are viewed by enrolled students.

UTOL also has a function that allows you to preview some types of course materials. By previewing them on UTOL, you can check the contents of files with the following extensions without having to download them to your device.

* File extensions: `.docx` / `.xlsx` / `.pptx`
* File extensions: `.pdf`
* File extensions: `.png` / `.gif` / `.jpg` / `.jpeg` / `.svg`

### Steps

1. Open the course top screen and click the ‚ÄúScreen for viewing‚Äù button at the top of the screen.

    ![](./06-preview-entry.drawio.png)

2. This will take you to the ‚ÄúScreen for viewing‚Äù. There, select the file you want to view from the ‚ÄúCourse Materials‚Äù column and click it.
3. A pop-up window will appear saying ‚ÄúPreview or download file‚Äù. Click the ‚ÄúView‚Äù button or the ‚ÄúDownload‚Äù button.
4. When you have finished viewing, click the ‚ÄúScreen for editing‚Äù button at the top of the screen to return to the editing page.

## Checking the course materials access status

This section explains how to check the access status of the course materials.

The course instructors and TAs can check which enrolled students have viewed the course materials and when they did it. The view status can be checked on UTOL. It can also be downloaded as an Excel file format (a password must be set for the file).

### Procedure

1. Open the ‚ÄúCourse Top‚Äù screen, and in the ‚ÄúCourse Materials‚Äù column, click the ![‚ãÆ](../../_icons/control_btn_icon.png){:.icon} button in the ‚ÄúActions‚Äù column to the right of the file name for which you want to check the viewing status.

    ![](./07-viewing-status-entry.drawio.png)

2. Click the ‚ÄúCheck view status‚Äù link.
3. A table showing the viewing status will be displayed.
    ![](./08-viewing-status.png)

4. To download in Excel file format:

    1. Click the ‚ÄúDownload‚Äù button.
    2. You will be asked to set a password for the Excel file. Enter a password and click the ‚ÄúDownload‚Äù button again.
