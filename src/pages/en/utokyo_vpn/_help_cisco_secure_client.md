  <summary>Troubleshooting guide: When the error message "Connect capability is unavailable because the VPN service is unavailable" appears</summary>
  
  Follow the steps below to allow the Cisco Secure Client to run in the background.
  1. Open "System Settings" in the Applications.
  2. Select "Login Items & Extensions" in "General" in "System Settings".
  3. If "Cisco Secure Client - AnyConnect VPN Service" in "App Background Activity" (or "Allow Apps to Run in the Background") is disabled, please enable "Cisco Secure Client - AnyConnect VPN Service" (click the toggle to the right of the item name and it turns blue).

  <div>If the issue is still not resolved, please consult the <a href="/en/support/">Technical Support Desk</a>.</div>
</details>
