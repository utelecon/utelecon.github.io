### Homebrewの利用

ECCSのMac環境では，[Homebrew (brew.sh)](http://brew.sh/) というパッケージシステムを使って，多数のフリーソフトを導入しています． ターミナルから，以下のコマンドを実行するとインストール済みのパッケージを確認することができます．

```
 brew list
```
