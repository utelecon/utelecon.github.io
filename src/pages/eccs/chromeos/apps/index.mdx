## 利用可能なアプリケーションについて

ChromeOSデバイスでは，Google Chromeとそのextensionとして実現されているアプリケーション，個別に許可された一部のAndroidアプリが利用可能です．なおAndroidアプリは一定の条件のもと[追加申請](#android-apps)が可能です．

また，一定の制約の元で[Linux環境](../linux/)を使うこともできます．

### プリインストールされているアプリケーションの一覧
{:#preinstalled-apps}

プリインストールされているアプリケーションは以下のとおりです．なお一部のアプリケーションについては，Progressive Web Appsと呼ばれる，Webサイトのアイコンををスマートフォンのネイティブアプリに近い形で表示する仕組みを利用しています．

- Files(ファイル): ECCSクラウドメールのアカウントに対応するGoogleドライブや，UTokyo AccountのOneDriveのファイルが利用可能です．詳しくは[ストレージに関するページ](../storage/)を参照してください．
- Calculator(電卓)
- Chrome: ブラウザとしてGoogle Chromeが利用可能です．
- Chrome Canvas(描画キャンバス)
- Gmail
- [Google Calendar](/google/calendar/)
- Google Chat
- Google Keep
- Google Maps
- Google ドライブ: 詳しくは[ストレージに関するページ](../storage/)を参照してください．
- Play Store(Play ストア): [許可されたAndroidアプリ](#android-apps)のインストールが可能です．
- Web Store(ウェブストア)
- Document(ドキュメント)
- Sheets(スプレッドシート)
- Slides(スライド)
- Text
- [Vids](https://support.google.com/docs/topic/14950067?hl=ja&ref_topic=1382883&sjid=13303492083534341122-NC): AIを利用した動画編集等ができます．
- [YouTube](/google/#youtube)
- [Zoom](/zoom/)
- Scan(スキャン): ただし[駒場の情報教育棟に実験的に設置しているScansnap](../../#ieb)での利用はできません．
- Camera(カメラ)
- Settings(設定)
- Print jobs(印刷ジョブ)
- [Cursive(手書きメモ)](https://support.google.com/chromebook/answer/11073444?hl=ja&sjid=13303492083534341122-NC)
- Key Shortcut(キーショートカット)
- Gallary(ギャラリー)
- [Screencast(スクリーンキャスト)](https://support.google.com/chrome/a/topic/15301726?hl=ja&ref_topic=4386930&sjid=13303492083534341122-NC): 録画などができます．
- Recorder
- [Explore(使い方・ヒント)](https://support.google.com/chromebook/answer/15470775?hl=ja)
- Terminal

## アプリの利用方法

### 開始

1. 左下のメニューを開いてください．
2. 起動したいアプリを選択してください．(上部の検索バーにアプリ名を入力して検索することもできます．)
    ![](./chrome_app_menu.png)

### 終了

アプリ右上の❎印を押してください．

## Androidアプリの追加申請
{:#android-apps}
